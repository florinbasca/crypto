# Signals

Cross-sectional statarb **spaces** — one economic hypothesis each — live in
`research/lib/spaces.py`. Add a space = add one `_S(...)` line there.

- `evaluate.py [name] [--no-save]` — score spaces' IC vs forward residuals.
- `signal_lab.py <name>` — KEEP/KILL scorecard (significance + orthogonality).
